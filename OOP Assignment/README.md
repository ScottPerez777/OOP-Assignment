# Python OOP Project

This is a simple 5-part project demonstrating basic Object-Oriented Programming concepts in Python.

## Setup

1. **Requirements:**  
   You'll need Python 3.x installed on your machine.

2. **How to Run:**

   - Save the project code in a file named `main.py`.
   - Open your terminal or command prompt, navigate to the project directory, and run:

   python main.py

   Usage:

Follow the on-screen prompts to see each part of the project in action. Each part will showcase class creation, methods, and object usage.
